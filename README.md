# music-scrobbling-web-app
A responsive music tracking web application built with HTML, CSS, JavaScript, AJAX, JSON, and PHP. The app allows users to view a music library, filter albums by year or genre, view listener statistics, add new listeners, and log song plays using asynchronous fetch requests.

## Features

- View albums from a JSON file
- Filter albums by year and genre
- View listener statistics
- Add new listeners with validation
- Log song plays using AJAX/fetch
- Display album images and listener avatars
- Responsive design for desktop and mobile

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AJAX / Fetch API
- JSON
- PHP

## What I Learned

Through this project, I practiced reading and updating JSON data, using asynchronous fetch requests, validating form input, sorting and filtering data, and designing a responsive user interface.
